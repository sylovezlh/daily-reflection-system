# 每日反思 · 使用指南

## 一、启动与访问

### 启动后端服务器

```bash
cd "/Users/sunyi/Desktop/cc/agent-skills包"
pip3 install flask        # 仅首次需要
python3 server.py
cd "/Users/sunyi/Desktop/每日反思系统"
python3 server.py
```

看到以下输出表示启动成功：

```
✦ 每日反思后端已启动 (含用户系统)
  主页面:    http://localhost:5001
  管理后台:  http://localhost:5001/admin
  API 就绪:  http://localhost:5001/api/ping
```

### 访问方式

| 场景 | 地址 | 说明 |
|------|------|------|
| 日常记录 | `http://localhost:5001` | 登录后填写反思 |
| 管理后台 | `http://localhost:5001/admin` | 管理员查看全站数据 |
| 离线使用 | 直接双击 `daily-reflection.html` | 无需服务器，数据存浏览器本地 |

### 关闭服务器

在终端按 `Ctrl + C` 即可停止。

## 二、账号系统

### 注册新账号

1. 打开 `http://localhost:5001`，自动弹出登录窗口
2. 点击 **「创建账号」** 标签
3. 输入用户名（2-30 个字符）和密码（至少 4 个字符）
4. 点击 **「创建账号」** 按钮

### 登录

1. 在 **「登录」** 标签下输入用户名和密码
2. 点击 **「登录」** 按钮

### 退出

点击页面标题栏右上角的 **「退出」** 按钮。

### 权限说明

| 角色 | 如何获得 | 权限 |
|------|---------|------|
| **管理员** | **第一个注册**的用户自动成为管理员 | 查看所有用户的记录 |
| **普通用户** | 后续注册的所有用户 | 只能查看自己的记录 |

## 三、日常记录

页面包含 **5 个板块**，建议每天晚上花 5 分钟填写：

### 1. 今日所做（左侧宽卡片）

记录今天完成的事情，**可设置时间段**用于统计占比。

**操作步骤：**
1. 点击卡片下方的「+ 点击添加」
2. 输入完成的事情，例如「完成了项目需求文档」
3. 点击条目左边的 `🕐 添加时间` 标签
4. 在弹出的面板中选择开始和结束时间：
   - **快捷时段**：早晨(06-08)、上午(08-12)、中午(12-13)、下午(13-18)、晚上(18-22)、深夜(22-06)
   - **自定义**：手动设置精确到分钟的起止时间
5. 点击 **「应用」**

> 设置了时间段的条目会自动计入右上角的**环形饼图**，直观显示各项任务的时间占比。

### 2. 处理的问题（右侧窄卡片）

记录今天遇到的困难和解决方法。

**提示：** 不一定是大问题，任何卡住过你的事情都值得记录。

### 3. 学习的技能（左下卡片）

记录今天学到的新知识、新工具、新方法。

**提示：** 可以是一行代码技巧、一个新快捷键、一个概念理解。

### 4. 今日之最（中央金色高亮卡片）

今天**最重要、最有价值**的一件事，会以金色高亮显示。

**提示：** 如果今天只能记住一件事，那是什么？

### 5. 不该做的事（底部全宽卡片）

诚实反思今天不该做的事情，值得改进的地方。

**提示：** 这个板块用玫瑰暗调设计，代表自我反思的氛围。

## 四、操作技巧

### 键盘快捷键

| 按键 | 行为 | 适用场景 |
|------|------|---------|
| `Enter` | 当前条目下方插入新行 | 任何输入框 |
| `Backspace` | 空条目时删除当前行 | 任何输入框 |
| `← →` | 切换前一天/后一天 | 非编辑状态 |
| `Escape` | 关闭弹窗 | 时间选择器、日期选择器 |

### 其他操作

| 操作 | 方式 |
|------|------|
| 切换日期 | 点击顶部日期文字，弹出**月历面板**快速跳转 |
| 回到今天 | 月历面板底部点击「回到今天」 |
| 删除条目 | 鼠标悬停在条目上，右侧出现 **×** 按钮 |
| 清除时间 | 时间面板中点击「清除」 |
| 快速定位板块 | 页面底部的导航圆点，点击跳转到对应板块 |

### 历史数据标记

月历中有记录的日期下方会显示**琥珀色圆点**，方便快速定位。

## 五、饼状图

### 显示条件

当「今日所做」板块中**至少有一个条目设置了开始和结束时间**时，右上角自动显示环形饼图。

### 图表内容

- **环形图**：每种颜色代表一个事件，面积 = 时长占比
- **中心数字**：当日总时长（例如 `5.5h`）
- **图例**：列出每个事件名称和百分比

### 实时更新

修改条目文字或调整时间后，饼图立即刷新。

## 六、管理后台（管理员专属）

### 进入方式

登录管理员账号后访问 `http://localhost:5001/admin`。

### 功能一览

| 功能 | 操作 |
|------|------|
| **查看全部用户** | 顶部下拉框选择「全部用户」或指定用户 |
| **查看统计** | 顶部卡片显示总用户数、总记录天数、总条目数 |
| **展开详情** | 点击某条记录旁的「展开」按钮查看完整内容 |
| **搜索** | 在搜索框输入关键词，按 `Enter` 搜索 |
| **删除记录** | 展开后点击「删除此日记录」 |

### 搜索示例

- 搜索「项目」→ 找出所有包含「项目」的条目
- 搜索「会议」→ 查看所有会议相关记录
- 搜索结果高亮显示匹配文本

## 七、同步机制

### 在线模式（有后端）

- 标题栏同步状态显示 **绿色圆点**
- 数据实时保存到 SQLite 数据库
- 换设备登录同一账号，数据完整同步

### 离线模式（无后端）

- 标题栏同步状态显示 **棕色圆点**
- 数据保存在浏览器 localStorage 中
- 功能完全正常，只是不能跨设备同步

### 数据安全

- 密码使用 PBKDF2-SHA256 + 随机盐哈希存储，不可逆
- 每次登录生成新的 64 位随机 token
- Token 失效后自动跳回登录页
- 不同用户的数据完全隔离

## 九、常见问题

**Q: 忘记了密码怎么办？**

目前需要管理员在服务器端重置。后续版本会加入密码重置功能。

**Q: 如何让别人也使用这个系统？**

对方访问 `http://你的IP:5001` 注册自己的账号即可，数据互不干扰。

**Q: 数据存储在哪里？**

- 在线：SQLite 数据库文件 `reflections.db`（与 server.py 同目录）
- 离线：浏览器 localStorage

**Q: 如何备份数据？**

复制 `reflections.db` 文件到安全位置即可完成备份。
